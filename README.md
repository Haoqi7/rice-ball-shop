# 家用饭团小铺 (Home Rice Ball Shop)

一个基于微信小程序原生框架和微信云开发实现的轻量级家庭点餐系统。专为家庭内部使用设计，支持多设备实时同步，无需配置复杂的服务器环境，即可实现“一人下单，全家同步”的趣味互动体验。

## 🌟 项目特色

- 零成本部署：基于微信云开发，无需购买服务器、域名或办理备案，完全免费（使用基础配额）。
- 实时同步：采用云数据库实时监听（Watch）技术，后厨端无需刷新即可感知新订单。
- 跨设备操作：支持多人同时在线，任何家庭成员都可以在不同设备上进行点餐、制作管理或菜单编辑。
- 温馨家用风：采用米黄色与浅棕色调设计，营造温馨的家庭厨房氛围。

## 🛠️ 技术栈

- 前端：原生微信小程序 (WXML, WXSS, JavaScript, JSON)
- 后端/存储：微信云开发 (WeChat Cloud Development)
  - 云数据库 (Cloud DB)：存储订单与菜品数据。
  - 实时监听 (Watch)：实现订单状态的秒级同步。
  - 用户授权：使用微信体验成员机制，保障家庭内部隐私。

## 📅 功能模块

### 1. 首页 (Index)

- 店铺名称与欢迎语展示。
- 快捷导航：下单、后厨、管理三大核心入口。
- 下单须知公告栏。

### 2. 下单模块 (Order)

- 菜品浏览：按“主食、炒菜、配料、饮品”分类展示，支持云端动态更新。
- 购物车：直观的增减操作，实时计算总数。
- 订单提交：支持添加备注（如：不要葱花），下单需输入家庭专属密码。
- 状态追踪：底部显示实时订单流，点击可进入详情页。

### 3. 详情页 (Order Detail)

- 可视化进度条：直观显示“待接单 -> 制作中 -> 已完成”三个阶段。
- 限时取消：支持在下单 5 分钟内且大厨未接单前自主取消订单。

### 4. 后厨模块 (Kitchen)

- 实时播报：利用实时监听功能，新订单自动弹出。
- 状态流转：一键标记“开始制作”或“制作完成”。
- 历史记录：查看已完成订单，支持一键清理云端历史数据。

### 5. 管理模块 (Admin)

- 菜单管理：异地同步增删改查菜品，支持分类设置。
- 安全设置：修改“下单密码”与“管理密码”。
- 数据维护：一键清空所有历史订单（保留菜单配置），方便定期重置测试数据。

## 🚀 部署指南 (5分钟快速上手)

1. 环境准备
   - 注册并登录 微信公众平台，获取你的 AppID。
   - 下载并安装 微信开发者工具。

2. 导入项目
   - 在开发者工具中选择“导入项目”，选择本项目代码文件夹，并填入你的 AppID。

3. 开通云开发
   - 点击工具栏左上角的“云开发”按钮，开通环境（建议选择免费基础版）。
   - 在“数据库”标签页中新建两个集合：
     - orders (用于存储订单)
     - menu (用于存储菜品)

4. 设置数据库权限 (关键)
   - 选中 orders 集合 -> 数据权限 -> 选择“所有用户可读写”（或使用自定义安全规则：{"read": true, "write": true}）。
   - 选中 menu 集合 -> 执行相同权限设置。

5. 初始化与发布
   - 在 app.js 中确认云开发环境初始化。
   - 点击“上传”按钮提交版本。
   - 在微信后台将该版本设为体验版。
   - 在“成员管理”中添加家人的微信号为体验成员。
